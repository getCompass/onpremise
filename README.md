## 🚀 Установка и настройка

В репозитории представлен исходный код приложения [Compass On-premise](https://getcompass.ru/on-premise).

Вы можете самостоятельно собрать его в контейнер Docker или использовать готовый контейнер
из [репозитория](https://github.com/getCompass/onpremise-installer), чтобы легко установить мессенджер
по [инструкции](https://doc-onpremise.getcompass.ru/?utm_source=github&utm_medium=downloads-and-settings&source_id=github).
